### Pull Request

1. 베이스 저장소 `Fork` 후 작업

2. 베이스 저장소에서 `Pull Requests` 들어가기

   <img src="../imgs/pr/01.png" width="500">

3. `New Pull Request` → `Compare Across Fork` → 내 저장소 선택

   <img src="../imgs/pr/02.png" width="500">

   <img src="../imgs/pr/03.png" width="500">

   <img src="../imgs/pr/04.png" width="500">

4. 하단에 보이는 수정 사항이 맞다면 `Create Pull Request`

   <img src="../imgs/pr/05.png" width="500">

5. 원하는 커밋 이름 작성 후 `Create Pull Request`

   <img src="../imgs/pr/06.png" width="500">

6. `Pull Request` 성공!

   <img src="../imgs/pr/07.png" width="500">

   - `Comment` 추가 가능
   - `Close Pull Request`로 풀리퀘 삭제 가능

     <img src="../imgs/pr/09.png" width="500">

7. 만약 베이스 저장소의 오너라면 `Merge Pull Request` 버튼이 보이며 풀리퀘한 수정 사항과 머지 가능
   <img src="../imgs/pr/08.png" width="500">
